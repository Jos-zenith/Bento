# Frontend

This folder contains the Next.js app router scaffold for Bento.

## Run locally

1. Install dependencies inside `frontend/`.
2. Start the dev server with `npm run dev`.
3. Open the app at `http://localhost:3000`.

## Included

- Next.js app router with TypeScript
- Responsive landing page
- Shared global styles
- Metadata and a clean project layout
# Frontend

This folder is reserved for the web client.

Recommended options:
- Next.js for the primary web app
- Flutter for mobile
- Streamlit for the initial technical demo
